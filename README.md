# Workflow Tracker

Workflow Tracker is a role-based task and workflow management system built using Spring Boot and PostgreSQL.

## Features
- Manager can create and assign tasks
- Users can accept tasks and update status
- Task comments visible to manager
- Role-based access control
- RESTful backend with basic frontend support

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA (Hibernate)
- Spring Security
- PostgreSQL
- Maven
- HTML, CSS, JavaScript
